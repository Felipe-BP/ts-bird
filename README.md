## Installation

Use the package manager [npm](https://www.npmjs.com) or [yarn](https://yarnpkg.com) to install dependencies.

```bash
npm install

or

yarn install
```

## Running

By default, snowpack server will run on http://localhost:8080.

```bash
npm run start

or

yarn start
```
